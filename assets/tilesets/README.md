Tilesets folder

Place the Dungeon tileset PNG and a Tiled JSON map here.

Recommended downloads (free):
- Dungeon tileset (16x16): https://opengameart.org/content/dungeon-tileset

Expected filenames:
- dungeon_tileset.png
- map.json  -> exported Tiled JSON referencing "dungeon_tileset.png"

If you don't provide these files the game will fall back to generated graphics.
